Week 4 - CIFAR-10 Image Classification using ANN and CNN

Overview :
This project focuses on image classification using the CIFAR-10 dataset. Three models were implemented and compared: a baseline Artificial Neural Network (ANN), a Convolutional Neural Network (CNN), and an Enhanced CNN with data augmentation and regularization techniques.

Key Tasks :
* Loaded and preprocessed the CIFAR-10 dataset
* Built and trained an ANN model
* Built and trained a CNN model
* Applied data augmentation techniques
* Implemented Early Stopping and Dropout
* Compared model performances using accuracy metrics

Results :
ANN - 41.14%
CNN - 68.46%
Enhanced CNN - 71.69%

Conclusion :
The results demonstrate that CNNs significantly outperform traditional ANNs for image classification tasks. Data augmentation and regularization further improved model performance and generalization.
