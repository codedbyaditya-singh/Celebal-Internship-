# Week 5 – Deep Learning Text Generation using RNN, LSTM & GRU

## Overview

This assignment implements a **Deep Learning-based text generation system** using recurrent neural network architectures. The models learn sequential patterns from a custom text corpus and generate meaningful text through next-word prediction.

## Implementations

- Text preprocessing and tokenization using **Keras Tokenizer**
- N-gram sequence generation and sequence padding
- **Vanilla RNN** model for text generation
- **LSTM** model for long-term sequence learning
- **GRU** model for efficient sequence modeling
- Training loss comparison of all three models
- Text generation using next-word prediction with **`np.argmax()`**
